Connected Backend Development for Game District's Shopify Store

Developed a connected backend application using Ruby on Rails for Game District’s Shopify store. The system integrates with both Shopify Admin and Storefront APIs to manage product listings and synchronize updates. Key features include:

  • Product Management: Automated product data handling, updates, and image synchronization across the store.
  • Data Scraping: Implemented a data scraping solution using Nokogiri and RestClient, populating the store with comprehensive product information gathered from external sources.

This solution enhanced the accuracy and efficiency of the store’s product management workflow, providing a seamless experience for both the store administrators and customers.
